Budgets App

This is a small project I'm doing using Django. Users can log in and add transactions they make, like purchases and income, and budgets to keep track of spending.

  • Writing front pages in HTML for users to access the site
  • Using Django's built-in functions on the front page to check if the user is logged in or not
  • Creating forms for the user to input transactions with details using Django's models and forms
  • Keeping up an SQL database and saving transactions, as well as queries when the user requests to see transactiond details

2-D Sidescroller using Unity Engine

Using the powerful Unity Enging, I'm creating a 2-D sidescroller game. The character moves across the screen and using C# scripts the game calls and listens for events.

  • Creating custom classes with built-in functionality
  • Using custom objects of those classes to interact with each other
  • Drawing every animation frame from scratch and connecting them to characters
  • Creating custom events that get invoked based on specific things that happen throughout the playthrough
  • Attaching event listeners to fire functions based on the events that happen
